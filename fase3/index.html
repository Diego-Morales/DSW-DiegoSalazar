Propósito

El propósito del sitio es implementar un recurso digital para dinamizar una estrategia de aprendizaje basada en los temas de etiquetas básicas de HTML, atributos y valores.

Introducción

HTML (HyperText Markup Language) es un lenguaje de programación que utiliza una serie de códigos llamados etiquetas, las cuales serán interpretadas por los navegadores de internet, estas van dando un formato y estructura al contenido de las páginas web.

Los atributos HTML se consideran como información adicional que se coloca siempre en la etiqueta de apertura, para añadir, complementar información o significado al elemento HTML y cuando se usa un atributo se debe especificar su valor, este valor representa una cantidad exacta, característica o la forma en que se ejecutará el atributo.
